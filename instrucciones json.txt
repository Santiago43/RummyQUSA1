
/*Cuando se conecta un nuevo usuario:*/

- Servidor retorna {tipo: "hash", hash: numero}

Cuando vaya a crear una sala:
- Cliente envía {tipo: "crear sala",usuario: nombre}
- Servidor retorna {tipo: "codigo sala", codigo: "código","mensaje": "Bienvenido nombre"}


Cuando vaya a conectarse a una sala:
- Cliente envia {tipo "conectarse a sala", codigo: "código",usuario: nombre}
- Servidor retorna {tipo: "conexión sala", mensaje: "conectado // error: esa sala no existe", participantes: numero}
- Servidor envía a todos los que estén en la sala {tipo: "nuevo jugador",participantes: numero}


Cuando vaya a desconectarse de una sala antes de jugar:

- Cliente envía {tipo: "desconectarse de sala"}
- Servidor envía a todos los que estén en la sala {tipo: "jugador salió", participantes: numero}



Cuando el jugador va iniciar partida: 

-Jugador inicia con {tipo: "iniciar partida"} //el servidor inicia el hilo sala
-Servidor revuelve y entrega fichas a cada jugador 

{
	tipo: "nueva mano",
	mano:[
			{
				numero: numero  , 
				color: color  ,
				x:  x  , 
				y:  y ,
				xAnterior:  xAnterior  ,
				yAnterior:  yAnterior 
			},
			{
				...
			},
			...,
			{
			...
			}
		]
	jugadores: [
			{nombre: nombreJugador},
			{nombre: nombreJugador2}
	]
}


Cuando el jugador coloca nueva ficha:

-Jugador envía
{
tipo: "jugada - colocar ficha", 
ficha: 
{
	numero: numero,
	color: numero,
	x: numero,
	y: numero,
	xAnterior:-1,
	yAnterior:-1
}
-Si el jugador está en turno, el servidor envía el dato de la ficha a todos los usuarios en la sala

-Si el jugador no está en turno, el servidor le retorna:

{
	tipo: "Error",
	mensaje: "tú no estás en turno"
}

Cuando el jugador mueve una ficha:

- Jugador envía

{
	tipo: "jugada - mover ficha",
	ficha: 
	{
		numero: numero,
		color: numero,
		x: numero,
		y: numero,
		xAnterior: numero,
		yAnterior: numero
	}
}

Cuando el jugador termina el turno (si realiza jugadas):

- Jugador envía 

{
	tipo: "fin de turno"
}

- Servidor responde si la jugada es válida o no

Si es válida

{
	tipo: "jugadas válidas"
}

Si no

{
	tipo: "jugadas inválidas"
}


Cuando el jugador roba ficha:

-Jugador envía
{
	tipo: "robar ficha"
}

-Servidor contesta
Si hay fichas en la banca
{
	tipo: "ficha robada",
	ficha:
	{
		numero: numero,
		color: numero,
		x: numero,
		y: numero,
		xAnterior: numero,
		yAnterior: numero
	}
}

Si no hay fichas en la banca

{
	tipo: "ficha no robada",
}
termina el turno.


Cuando se cambia de turno el servidor manda mensajes a todos:

{
	tipo: "mensaje del servidor",
	mensaje: "el mensaje"
}

Cuando indica inicio de turno, se lo envía al jugador:

{
	tipo: "cambio turno",
	valor: boolean
}

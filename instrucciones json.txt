
/*Cuando se conecta un nuevo usuario:*/

- Servidor retorna {tipo: "hash", hash: numero}
- Cliente contesta {tipo: "nuevo",usuario: nombre}
- Servidor contesta {tipo: "nueva", "mensaje": "Bienvenido nombre"}

Cuando vaya a crear una sala:
- Cliente envía {tipo: "crear sala"}
- Servidor retorna {tipo: "codigo sala", codigo: "código"}


Cuando vaya a conectarse a una sala:
- Cliente envia {tipo "conectarse a sala", codigo: "código"}
- Servidor retorna {tipo: "conexión sala", mensaje: "conectado // error: esa sala no existe"}
- Servidor envía a todos los que estén en la sala {tipo: "nuevo jugador"}
- Servidor envía al usuario {tipo: "participantes", participantes: numero}


Cuando vaya a desconectarse de una sala antes de jugar:

- Cliente envía {tipo: "desconectarse de sala"}
- Servidor envía a todos los que estén en la sala {tipo: "jugador salió"}


Cuando está en espera de partida:

-Servidor envía {tipo: "participantes", participantes: numero}


Cuando el jugador va iniciar partida: 

-Jugador inicia con {tipo: "iniciar partida"} //el servidor inicia el hilo sala


Cuando el jugador coloca nueva ficha:

-Jugador envía 
{
tipo: "jugada - colocar ficha", 
ficha: 
{
	numero: numero,
	color: numero
},
coordenada: {
		x: numero,
		y: numero
	}	
}


Cuando el jugador mueve una ficha:

- Jugador envía

{
	tipo: "jugada - mover ficha",
	ficha: 
	{
		numero: numero,
		color: numero
	},
	coordenada_anterior: 
	{
		x: numero,
		y: numero
	},
	coordenada_nueva: 
	{
		x: numero,
		y: numero
	}
}

Cuando el jugador termina el turno:

- Jugador envía 

{
	tipo: "fin de turno"
}

- Servidor responde si la jugada es válida o no

Si es válida

{
	tipo: "jugadas válidas"
}

Si no

{
	tipo: "jugadas inválidas"
}
